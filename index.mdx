## Get started

<Steps>
  <Step title="Install the CLI">
    Install `@devcontainers/cli` globally via npm or use the standalone install script:
    ```bash
    npm install -g @devcontainers/cli
    ```
  </Step>
  <Step title="Add a devcontainer.json">
    Create a `.devcontainer/devcontainer.json` in your project:
    ```json
    {
      "image": "mcr.microsoft.com/devcontainers/base:ubuntu",
      "features": {
        "ghcr.io/devcontainers/features/node:1": {}
      }
    }
    ```
  </Step>
  <Step title="Start the container">
    Run `devcontainer up` in your project directory:
    ```bash
    devcontainer up --workspace-folder .
    ```
  </Step>
  <Step title="Execute commands inside">
    Run tools or scripts inside the running container:
    ```bash
    devcontainer exec --workspace-folder . node --version
    ```
  </Step>
</Steps>

<Note>
  The Dev Containers CLI is part of the [Development Containers Specification](https://containers.dev/). It works with Docker, Docker Compose, and supports the full `devcontainer.json` configuration surface.
</Note>
